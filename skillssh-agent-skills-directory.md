# The Agent Skills Directory

**Source:** https://skills.sh/

## Overview

The Agent Skills Directory is an open ecosystem platform for discovering and installing reusable AI agent capabilities. Skills are procedural knowledge packages that enhance AI agents with specific expertise and best practices. The platform enables one-command installation across multiple agent environments.

## Key Topics

### Installation

Skills can be added using a single command:

```bash
npx skills add <owner/repo>
```

This streamlined approach allows developers to quickly enhance their agents with new capabilities without manual configuration.

### Supported Agents

The directory supports over 20 different AI agent platforms including:

- Claude Code
- GitHub Copilot
- Cursor
- Gemini
- And many others

### Popular Skills Categories

Skills cover diverse domains:

- **Frontend frameworks** - React, Vue, Next.js best practices
- **Backend development** - Node.js, FastAPI, NestJS patterns
- **DevOps and deployment** - CI/CD, infrastructure patterns
- **Marketing and content** - Content strategy guidelines
- **Security and testing** - Security best practices, testing patterns
- **Design and UX** - Design systems, UX patterns

### Leaderboard System

The platform ranks skills by installation metrics:

- All-time popularity rankings
- Trending skills (24-hour basis)
- Top skills include "vercel-react-best-practices" and "web-design-guidelines"

## Use Case

The directory serves as a centralized marketplace where developers can discover vetted agent capabilities, reducing setup complexity and enabling consistent best practices across AI-assisted development workflows.
